# Predictive-Water-Resource-Modeling-Arid-Grove-Eco-Resort---Desert-Water-Demand-
Objective: Build a regression model to forecast daily water demand (gallons/day) to optimize water-hauling schedules and greywater recycling operations.
Data Sources:
Hotel Booking Demand Dataset(Kaggle / Jesse Mostipak, CC0 Public Domain),
Weather Data (Open-Meteo)
Models Evaluated:
Elastic Net | Random Forest Regression | XGBoost
Key Engineered Features
Peak Heat Days (≥95℉) | High-Density Booking Periods (>70% occ.)
